# Autonomous_Research_Agent
An autonomous AI agent capable of collecting information from external sources, analyzing it, and generating a structured, actionable summary.  

Requirements:  
The agent should:  
 Accept a user query/topic.  
 Search one or more external sources (websites, APIs, documentation, etc.).  
 Extract relevant information.  
 Remove duplicate or irrelevant content.  
 Generate a well-structured summary containing:  
o Key points  
o Important findings  
o References/Sources  
o Actionable insights (if applicable)  

Bonus:  
 Allow the LLM to autonomously select the most appropriate external  
data sources based on the users query.  
 Parallel information gathering from multiple sources.  
 Export summary as PDF or Markdown.  
 Store previous searches in memory.
