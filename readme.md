# Autonomous AI Research Agent

## Features

- Autonomous planning
- Multi-source search
- Parallel retrieval
- Duplicate removal
- Relevance ranking
- LLM summarization
- PDF export
- Markdown export
- Memory
- FastAPI backend

## Installation

```bash
git clone <repo>

cd autonomous-ai-agent

pip install -r requirements.txt
```

Create .env

Run

```bash
uvicorn app:app --reload
```
